# 银行信用卡业务竞品监控系统

基于 Apple App Store 公开数据的银行信用卡竞品监控工具。实时追踪 6 家头部银行信用卡 App 的评分变化、营销活动、用户口碑和版本迭代，自动生成周度竞品分析报告，为商务谈判和策略制定提供数据支撑。

## 功能模块

| 模块 | 功能 | 技术 |
|------|------|------|
| 📡 数据采集 | App 元数据、评分、评论自动抓取 | iTunes Search / Lookup API + RSS |
| 📊 评分追踪 | 评分变化趋势、异动预警 | 历史对比 + 阈值告警 |
| 💬 情感分析 | 评论情感打分 + 好评/差评关键词提取 | jieba 分词 + SnowNLP |
| 🏦 竞品矩阵 | 多维度排名对比（评分/评论量/好评率） | 聚合计算 |
| 🎁 **信用卡活动监测** | 6 大类活动关键词识别（新户礼/分期/积分/权益/支付/联名卡） | 评论 NLP + 关键词规则 |
| ⚠️ **信用卡风险口碑** | 5 维度负面评论自动预警（发卡/权益/客服/积分/活动套路） | 风险关键词 + 阈值告警 |
| 📋 可视化报告 | ECharts 交互式看板（评分/活动/风险三区） | Jinja2 + ECharts 5.5 |
| 📅 周度竞品报告 | 评分趋势+活动热度+风险预警+商务策略建议 | 自动生成 |
| 🔔 自动通知 | 飞书/钉钉 Webhook 推送 | Webhook |

## 快速开始

```bash
pip install -r requirements.txt

# 一键执行：采集 → 分析 → 活动分析 → 报告
python main.py run

# 周度竞品分析报告
python main.py weekly

# 单独执行各步骤
python main.py crawl      # 采集 App Store 数据
python main.py activity   # 信用卡活动 + 风险口碑分析
python main.py report     # 生成 HTML 看板
python main.py notify     # 异常预警推送
```
